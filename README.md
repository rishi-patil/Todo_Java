## PREREQUISITES
- Maven 3+
- Java 8+

## BOOTING UP
For simplicity's sake, I prefer to use the command line when trying something new but you can google on how to set up a maven project in your favorite IDE.. ðŸ˜‰
<ol>
<li>Clone the repository:</li>
  <code>$ git clone https://github.com/davidarchanjo/spring-boot-todo-crud</code>
<li>Navigate to the folder:</li>
  <code>$ cd spring-boot-todo-crud</code>
<li>Run the application:</li>
  <code>$ mvn spring-boot:run</code>
</ol>

If the last command ran well, go to http://localhost:8080 and check the application running!
